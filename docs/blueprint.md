## Core Features:

- Product Catalog: Display pizzas and other products in organized categories.
- Order Assembly: Allow customers to assemble their order with options for pizza size, crust, and additional notes.
- WhatsApp Order Submission: Generate and send the order to the pizzeria's WhatsApp.
- Admin Product Management: Enable administrators to add, edit, and remove products with images and details.
- Order Visualization: Enable admin to see a real-time overview of the orders
- App Personalization: Allow administrators to change the pizzeria's name, logo, and main colors.

## Style Guidelines:

- Primary color: Vivid red (#FF4136) to evoke the richness of tomato and appetite appeal.
- Background color: Desaturated red (#F2D7D5) to offer a complementary, subtle backdrop.
- Accent color: Golden yellow (#FFD700) to suggest the creaminess of cheese, enhancing visual interest.
- Body and headline font: 'Poppins' for a modern, geometric sans-serif style that enhances readability.
- Use icons that are clear, related to food, and suitable for a general audience, especially older users.
- Design a straightforward, spacious layout with large buttons, maximizing usability for people of all ages.
- Include animations that enhance usability without distracting users. For example, subtle animations on hover or loading states.
